Sobre l'Exploració Espacial

L'exploració espacial és l'ús de l'astronomia i la tecnologia espacial per explorar l'espai exterior. L'exploració física de l'espai es realitza tant per vols espacials tripulats com per naus espacials robòtiques.

El Nostre Sistema Solar

Mercuri

El planeta més petit i més interior del Sistema Solar.

Venus

El segon planeta des del Sol i el veí més proper de la Terra.

Terra

El nostre planeta natal, l'únic planeta conegut que alberga vida.

Mart

El quart planeta des del Sol, sovint anomenat el "Planeta Vermell".

Júpiter

El planeta més gran del nostre Sistema Solar.

Saturn

El sisè planeta des del Sol, famós pels seus impressionants anells.

Urà

El setè planeta des del Sol, conegut pel seu to blau-verdós i el seu eix inclinat.

Neptú

El vuitè i més llunyà planeta del nostre Sistema Solar, amb un intens color blau.

Tutorial: Com descarregar vídeos amb youtube-dl

Pas 1: Instal·lar youtube-dl

# Per a Linux/MacOS
pip install youtube-dl

# Per a Windows (amb pip)
pip install youtube-dl

També pots descarregar l'executable directament des de youtube-dl.org.

Pas 2: Descarregar un vídeo

youtube-dl https://www.youtube.com/watch?v=Ilifg26TZrI

Això descarregarà el vídeo amb la millor qualitat disponible.

Pas 3: Descarregar només l'àudio

youtube-dl -x --audio-format mp3 https://www.youtube.com/watch?v=Ilifg26TZrI

Això extraurà només l'àudio del vídeo i el guardarà en format MP3.

Pas 4: Descarregar amb una qualitat específica

youtube-dl -f 22 https://www.youtube.com/watch?v=Ilifg26TZrI

El format 22 correspon a vídeo MP4 a 720p. Per veure tots els formats disponibles: 

youtube-dl -F https://www.youtube.com/watch?v=Ilifg26TZrI

Tutorial: Convertir vídeos a diferents formats amb FFmpeg

Pas 1: Instal·lar FFmpeg

# Per a Ubuntu/Debian
sudo apt-get install ffmpeg

# Per a MacOS (amb Homebrew)
brew install ffmpeg

# Per a Windows
# Descarrega des de https://ffmpeg.org/download.html

Pas 2: Convertir un vídeo a MP4

ffmpeg -i video_entrada.mov video_sortida.mp4

Pas 3: Convertir a WebM (format per a web)

ffmpeg -i video_entrada.mov -c:v libvpx-vp9 -crf 30 -b:v 0 video_sortida.webm

Pas 4: Extraure l'àudio d'un vídeo

ffmpeg -i video_entrada.mp4 -q:a 0 -map a audio_sortida.mp3

Pas 5: Comprimir un vídeo

ffmpeg -i video_entrada.mp4 -vf scale=1280:-1 -c:v libx264 -crf 28 -preset medium -c:a aac -b:a 128k video_comprimit.mp4

Aquest comandament redimensiona el vídeo a 1280px d'amplada i comprimeix tant el vídeo com l'àudio.
